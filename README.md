# PINGOBRAS
A Pingobras S.A é uma empresa que se dedica a hospedar e criar sites ou servidores, além de oferecer serviços de empréstimos. Atualmente, a empresa pode ser acessada através do endereço https://pingobras.glitch.me.

Modernização
------------
Este repositório contém a versão original do site (2019/2020). O site está obsoleto e há um projeto em andamento para modernizá-lo usando Bootstrap, organização de `src/js/` e `src/css/`. Veja o roteiro e checklist em [MODERNIZATION.md](MODERNIZATION.md).

Requisitos da empresa (resumo)
------------------------------
- Paleta principal: azul e roxo.
- Animações: animações suaves em todas as páginas, priorizando performance.
- Objetivo: apresentar a empresa Pingobras S.A. e centralizar todos os sistemas web.
- Segurança: sempre priorizar — usar HTTPS, validação de entradas e proteção das rotas da API.
- API atual: `https://pingobras-sg.onrender.com/` com rota base `/api`.
- Páginas de funcionários: manter sub-sites pessoais/ blogs integrados (ex.: `LUIS_DAS_ARTIMANHAS/`).
